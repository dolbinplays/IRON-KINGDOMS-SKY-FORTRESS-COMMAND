# IRON KINGDOMS: SKY FORTRESS COMMAND

Version: v0.26.06.23.0006

Open `index.html` directly in a browser. This build keeps the game as a single-file HTML experience using the existing Three.js CDN reference plus matching GLTFLoader, with MP3 music assets under `assets/audio/music/` and Brasswake GLB model parts under `assets/models/brasswake/`.

V3 canon now starts the campaign aboard the inherited Tier 0 skiff `Brass Minnow`. Brasswake remains a locked endgame fortress-city milestone and future home of Brasswake Free Command.
