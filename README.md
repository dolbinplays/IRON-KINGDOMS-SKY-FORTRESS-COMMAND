# IRON KINGDOMS: SKY FORTRESS COMMAND

Version: v0.26.06.26.0001

Open `index.html` directly in a browser. This build remains a single-file HTML experience using the existing Three.js CDN reference plus matching GLTFLoader, with MP3 music assets under `assets/audio/music/` and Brasswake GLB model parts under `assets/models/brasswake/`.

V3 canon starts the campaign aboard the inherited Tier 0 skiff `Brass Minnow`. Brasswake remains a locked endgame fortress-city milestone and future home of Brasswake Free Command.

This patch replaces the active player craft visual with a simple modular Brass Minnow placeholder skiff: compact hull, rear engine block, one smokestack, lift pods, tail rudder, deck/cargo plate, small utility arm, lantern, and patch plates. `SAVE_KEY` intentionally remains `IK_SKY_FORTRESS_COMMAND_v0_26_06_24_0001` because this patch does not change save-state structure.
