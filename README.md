# IRON KINGDOMS: SKY FORTRESS COMMAND

Version: v0.26.06.23.0005

Open `index.html` directly in a browser. This build keeps the game as a single-file HTML experience using the existing Three.js CDN reference plus matching GLTFLoader, with MP3 music assets under `assets/audio/music/` and Brasswake GLB model parts under `assets/models/brasswake/`. The Brasswake GLB import path sanitizes embedded cameras, lights, helpers, and debug/icon artifacts before model parts are placed.
